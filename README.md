# claude-md templates

Готовые **CLAUDE.md-шаблоны** для проектов, которые ведутся вместе с Claude. Каждый — самодостаточная схема «LLM-вики»: структура знаний, операции (ingest / query / lint) и дисциплина, не дающая знанию гнить.

## Шаблоны

- **`saas-product/`** — разработка SaaS / программного продукта: спеки, спринты, архитектура (`implementation:`-связь с кодом), product-discovery.
- **`research-project/`** — исследование без кода: центральный тип `claims/` с цепочкой доказательств, «источники побеждают вики», локализация цитат.
- **`business-run-general/`** — ведение бизнеса и личных проектов: контрагенты (`entities/`), календарь обязательств, роли-линзы.
- **`base-template/`** — общий скелет, из которого собраны остальные. Не самостоятельный (содержит пустые слоты `<<СЛОТ S#>>`); смотрите его, чтобы понять общую механику.

## Как использовать

1. Скопируйте содержимое нужного каталога в корень своего репозитория.
2. Заполните раздел «О проекте» под себя — остальное наследуется без изменений.
3. Запускайте `lint` периодически: проект сам сверится с обновлениями шаблона здесь (поле версии в шапке `CLAUDE.md` + `CHANGELOG.md`) и **предложит** подтянуть новое, не затирая ваши правки.

## Про этот репозиторий

Это **зеркало**: публикуется автоматически из приватного репозитория-источника, где живёт методология и история разработки шаблонов. Сюда не нужно слать PR — обновления приходят в вашу копию через `lint` на вашей стороне.
