# CHANGELOG — saas-product

Версия шаблона = номер верхней записи. Растёт на +1 при изменении, которое стоит распространять в проекты, инстанцированные из этого шаблона; мелкие правки формулировок версию не двигают. Теги: `[base]` — из общего скелета base-template, `[saas]` — собственная специфика класса.

Выровнен с **base-template@v1**.
