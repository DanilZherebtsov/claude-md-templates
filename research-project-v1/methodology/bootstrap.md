# Bootstrap — инициализация под конкретное исследование

Запускается при первом ingest в проект без структуры или по просьбе («инициализируй структуру», «разверни вики»). **Никогда** из `query`/`lint`: нет структуры — Claude просит сначала что-нибудь дать. Один раз при старте; повторно — только восстановление разрушенного.

Bootstrap — короткая настройка под исследование. Шаблон уже содержит `CLAUDE.md`, `STATE.md`, `methodology/`; bootstrap достраивает рабочие слои.

## Шаг 1 — короткое интервью (3–5 вопросов)

- Главный исследовательский вопрос и аудитория результата?
- Какие источники будут приходить (литература, полевые данные, переписка с экспертами)?
- Будут ли собственные датасеты (`data/`) или код анализа (`src/`)?

## Шаг 2 — предложить структуру

Claude предлагает раскладку `raw/` под названные источники — по умолчанию стандартный набор `literature/`, `fieldwork/`, `conversations/`, `decisions/` (+ `datasets/`, если есть свои данные). Человек подтверждает/правит.

## Шаг 3 — создать (чего нет, не перезаписывая существующее)

- `raw/.gitkeep` + согласованные подпапки.
- `wiki/` + 4 type-папки (`claims/`, `decisions/`, `synthesis/`, `principles/`, каждая с `.gitkeep`); `wiki/index.md` (`# Wiki Index`); `wiki/log.md` (`# Wiki Log` + `## [YYYY-MM-DD] bootstrap | инициализация`).
- `output/.gitkeep` + `output/drafts/.gitkeep` (рабочие файлы вопросов `q-NNN-...` кладутся прямо в `output/` по факту).

## Шаг 4 — заполнить сопутствующие файлы из ответов интервью

- «О проекте» в `CLAUDE.md` (главный вопрос, аудитория, рамки).
- **Дерево «Архитектура» в `CLAUDE.md`** — отразить созданные подпапки `raw/`, если отличаются от стандартных.
- Стартовый раздел `STATE.md` (стадия); `_Обновлено:_` = сегодня.

## Что НЕ создаётся при bootstrap

- `src/`, `data/` — по решению человека, когда нужны (код анализа; собственные датасеты).
- Подпапки `raw/` сверх согласованных — по факту (правило «Структура растёт по мере надобности» в [CLAUDE.md](../CLAUDE.md)).

После bootstrap — если проект под git — один коммит `bootstrap: инициализация структуры`; без git шаг пропускается.
